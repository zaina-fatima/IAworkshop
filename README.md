# Image Analysis workshop May 2026 (Padova, Italy)
## Instructor: Stefania Marcotti, Francis Crick Institute

This repository contains selected exercises completed during the Image Analysis Workshop held at Università degli Studi di Padova (27–29 May 2026).

## Included Exercises
1. Batch Image Analysis

Python workflow for:

loading multiple images from a folder
automated segmentation
object counting
extraction of morphology measurements such as area and eccentricity
storing results in pandas dataframes

2. 3D Object Counting

Analysis of 3D microscopy datasets using Python and napari, including:

visualization of volumetric image data
segmentation of nuclei/objects in 3D
quantitative object counting

## Tools & Libraries
Python
Jupyter Notebook
NumPy
Pandas
scikit-image
Matplotlib
napari

## Purpose

The goal of these exercises is to practice reproducible and automated image analysis workflows for biological imaging datasets.
